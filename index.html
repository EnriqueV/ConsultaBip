ConsultaBip!

Firefox OS App para consultar el Saldo de la tarjeta Bip!, medio de pago del transporte público de la ciudad de Santiago de Chile.

Download this project as a .zip file Download this project as a tar.gz file

¡Bienvenido al proyecto ConsultaBip!.

Este proyecto es parte de uno más grande, una suite de apps utilitarias para los servicios públicos de Santiago de Chile (bip!, metro, transantiago, etc).

Para obtener los fuentes sólo basta con un:

$ git clone https://github.com/Psep/ConsultaBip.git

A pesar de que el template de la web recomienda otras aplicaciones, usemos la shell, no cuesta mucho ;-)

¿De dónde se obtiene la data?

La información del aplicativo se obtiene directamente de la web de consultas de la tarjeta bip. Se ha construido una pequeña api que es parte de otro proyecto personal para generar múltiples APIs públicas para consumir apps web de servicios llamada "apis-services" (nombre original (?)). Puedes revisar el repositorio aquí.
